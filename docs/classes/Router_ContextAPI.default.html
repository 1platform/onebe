Class default<BodyRequest>

Endpoint request Context information class.

Through this class you can interact with the data received from your users. This Context API allows you to get data from the Parameters, Query Parameters or Body easily, through specific methods and specific data types.

  • Context API constructor.

    Type Parameters

    • BodyRequest = any

    Parameters

    • request: Request<ParamsDictionary, any, any, ParsedQs, Record<string, any>>

      The request object for which you need context.

    • response: Response<any, Record<string, any>>

      The response object for which you need context.

    • Optional exposeRequest: boolean = false

      Flag to let the Context API know if you need the request object exposed in the endpoint method or not.

    • Optional isGet: boolean = false

      Flag to let the Context API know that the request is a GET request.
